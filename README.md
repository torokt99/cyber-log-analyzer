
# Cyber Log Analyzer

A cybersecurity-focused log analysis tool built with FastAPI.

## Features

- Parse authentication logs
- Extract source IP addresses
- Detect brute-force login attempts
- REST API endpoints

## Technologies

- Python
- FastAPI
- Regular Expressions
- Git / GitHub

## API

### GET /

Returns API status.

### GET /alerts

Returns suspicious IP addresses detected from log files.
